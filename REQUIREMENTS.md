# 作業管理タイマー (Work/Free Timer) 要件定義書

## 概要
仕事と自由の時間を交互に管理するためのiOSタイマーアプリ。

## 機能要件

### 1. タイマーモード
- **仕事モード (Work Mode)**
    - 設定された時間からカウントダウンを行う。
    - 0秒になっても停止せず、継続して時間を計測する（延長時間のカウント）。
    - 延長時はカウントがマイナス表示になり、文字色がオレンジに変更される。
    - 終了時のアラート通知は行わない。
- **自由モード (Free Mode)**
    - 設定された時間からカウントダウンを行う。
    - 0秒になった時点で停止する。

### 2. 自動移行機能
- 自由モードのカウントが終了（0秒）した際、即座に仕事モードへ自動移行する。
- 移行時にシステム通知音（ビープ音）を鳴らす。
- ユーザーによる確認（アラートのOKボタン押下など）は不要。

### 3. 設定・カスタマイズ
- **時間の編集**
    - 仕事時間および自由時間を、分単位・秒単位で個別に設定可能。
    - 設定画面（ギアアイコン）から変更を行う。
- **手動コントロール**
    - タイマーの開始（Play）/ 一時停止（Pause）。
    - タイマーのリセット（現在のモードの初期値に戻す）。
    - モードの手動切り替え。

### 4. ユーザーインターフェース (UI)
- モード表示（Work / Free）。
- デジタル時計形式のカウント表示。
- 操作ボタン（Play/Pause, Reset, Switch Mode）。
- 設定用アイコン（Gear）。

## 非機能要件
- **プラットフォーム**: iOS (SwiftUI)
- **実行環境**: iPhone シミュレーター
- **バージョン管理**: GitHub (`fe_modoki` リポジトリ)
