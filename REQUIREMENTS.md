# 作業管理タイマー (Work/Free Timer) 要件定義書 - v1.3.0

## 1. プロジェクト概要
「１５分チェンジ法」（参考: [集中力を高めるやり方① : Ixyのお絵かき情報ブログ](https://ixy.blog.jp/archives/2366349.html)）をベースとした、集中力をコントロールするためのiOSアプリ。
「好きなこと」と「やるべきこと」を交互に繰り返すことで、作業への心理的障壁を下げ、自然な集中状態（フロー）へ導くことを目的とする。

## 2. 機能要件

### 2.1 タイマーモード
- **共通**: 0:00から目標時間に向かって進む**カウントアップ方式**。
- **仕事モード (Work Mode / やるべきこと)**
    - 目標時間を経過しても停止せず、**継続して計測（延長）**を行う。
    - 「ノってきた」場合はそのまま作業を継続することを推奨する仕様。
    - 目標時間を超過した際は、タイム表示が**オレンジ色**に変更される。
    - 目標時間到達時のアラート通知は、作業を妨げないよう**一度のみ**鳴動（設定でONの場合）。
- **自由モード (Free Mode / 好きなこと)**
    - 目標時間に達した時点でカウントが**停止**する。
    - 目標時間到達時に、停止ボタンを押すまで**アラートが鳴り続ける**。これにより強制的に作業の切り替えを促す。

### 2.2 自動移行・開始機能
- **アラート停止後の連動**: 
    - 自由モードのアラートを停止（STOP）した際、即座に**「仕事モード」へ自動切り替え**し、タイマーを**自動スタート**させる。
    - 仕事モードのアラート（単発）が発生している場合に操作を行った際も、同様に「自由モード」への切り替えと自動スタートを行う。

### 2.3 記録・履歴機能
- **セッション記録**:
    - モード切り替え（手動・自動問わず）が行われた際、そのセッションの種類と実際の経過時間を記録。
- **履歴表示**:
    - メイン画面のリストアイコンから過去50件分程度の履歴を確認・削除可能。

### 2.4 設定・カスタマイズ
- **時間の編集**: 仕事・自由それぞれの目標時間を「分」「秒」単位で設定可能。
- **通知設定**: 仕事モードでのアラート通知の有無を切り替え可能。
- **永続化**: 設定および履歴データはアプリ終了後も保持される。

### 2.5 ユーザーインターフェース (UI)
- **メイン画面**: モード名、カウント、統合された操作ボタン（開始/一時停止/アラート停止、リセット、切替）。
- **バージョン表示**: v1.2.0 （または v1.3.0）を明記。
