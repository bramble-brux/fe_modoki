# 作業管理タイマー (Work/Free Timer) 要件定義書

## 概要
仕事と自由の時間を交互に管理するためのiOSタイマーアプリ。

## 機能要件

### 1. タイマーモード
- **仕事モード (Work Mode)**
    - 設定された時間に向けてカウントアップを行う。
    - 設定された時間になっても停止せず、継続して時間を計測する（延長時間のカウント）。
    - 延長時は文字色がオレンジに変更される。
    - 目標時間到達時のアラート通知は一度だけ行われる（設定でONの場合）。
- **自由モード (Free Mode)**
    - 設定された時間に向けてカウントアップを行う。
    - 設定された時間になった時点で停止する。
    - 終了時のアラート通知は停止ボタンを押すまで鳴り続ける。

### 2. 自動移行機能
- 自由モードのカウントが終了（0秒）した際、自由モードを停止するまではアラートが鳴り続ける。
- 自由モードを停止すると、仕事モードに移行し、タイマーが自動でスタートする。

### 3. 設定・カスタマイズ
- **時間の編集**
    - 仕事時間および自由時間を、分単位・秒単位で個別に設定可能。
    - 設定画面（ギアアイコン）から変更を行う。
- **手動コントロール**
    - タイマーの開始（Play）/ 一時停止（Pause）。
    - タイマーのリセット（現在のモードの初期値に戻す）。
    - モードの手動切り替え。

### 4. ユーザーインターフェース (UI)
- モード表示（Work / Free）。
- デジタル時計形式のカウント表示。
- 操作ボタン（Play, Switch Mode）。
- 設定用アイコン（Gear）。

### 5. 記録機能

- [x] 終了時に作業時間と自由時間の合計をそれぞれ計測する
- [x] 日々の記録して振り返れる

## 非機能要件
- **プラットフォーム**: iOS (SwiftUI)
- **実行環境**: iPhone シミュレーター
