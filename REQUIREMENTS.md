# 作業管理タイマー (Work/Free Timer) 要件定義書 - v1.2.0

## 1. プロジェクト概要
仕事（集中）と自由（休憩）のサイクルを効率的に管理するためのiOSアプリ。
タイマーの挙動、自動移行、および実績の記録により、ユーザーのワークフローをサポートする。

## 2. 機能要件

### 2.1 タイマーモード
- **カウント方式**: 全てのモードで「0:00」から目標時間に向かって進む**カウントアップ方式**。
- **仕事モード (Work Mode)**
    - 目標時間を経過しても停止せず、**継続して計測（延長）**を行う。
    - 目標時間を超過した際は、タイム表示が**オレンジ色**に変更される。
    - 目標時間到達時のアラート通知は**一度のみ**鳴動する（設定でONの場合）。
- **自由モード (Free Mode)**
    - 目標時間に達した時点でカウントが**停止**する。
    - 目標時間到達時に、停止ボタンを押すまで**アラートが鳴り続ける**。

### 2.2 自動移行・開始機能
- 自由モードのアラートを停止（STOP）した際、即座に**「仕事モード」へ自動切り替え**を行う。
- 切り替えと同時に、仕事モードのタイマーが**自動的にスタート**する。

### 2.3 記録・履歴機能
- **セッション記録**:
    - 手動でのモード切り替え時、または自由モード終了に伴う自動移行時に、そのセッションの種類（Work/Free）と実際の経過時間を記録する。
- **履歴表示**:
    - メイン画面のリストアイコンから過去50件分程度の履歴を確認できる。
    - 履歴の個別削除が可能。

### 2.4 設定・カスタマイズ
- **時間の編集**: 仕事・自由それぞれの時間を「分」および「秒」単位で設定可能。
- **通知設定**: 仕事モードでのアラート通知の有無を切り替え可能。
- **データの永続化**:
    - 目標時間の設定、通知設定、および履歴データは**アプリ終了後も保持**される。

### 2.5 ユーザーインターフェース (UI)
- **メイン画面**:
    - 現在のモード名とタイマー表示。
    - 統一された操作ボタン（開始/一時停止/アラート停止、リセット、モード切替）。
    - 履歴/設定へのナビゲーション。
- **バージョン表示**: 画面最下部に「v1.2.0」を表示。
