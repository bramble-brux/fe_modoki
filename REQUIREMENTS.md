# 作業管理タイマー 要件定義書 - v1.5.0

## 1. 概要
「１５分チェンジ法」（参考: [Ixyブログ](https://ixy.blog.jp/archives/2366349.html)）に基づく集中力管理アプリ。
「好きなこと（Free）」と「やるべきこと（Work）」を交互に繰り返し、自然な集中状態へ導く。

## 2. タイマーモード

### 2.1 共通仕様
- 0:00から目標時間へ向かう**カウントアップ方式**。
- ボタンは常に**1つだけ**。押すと反対モードに切り替わり自動スタート。

### 2.2 仕事モード (Work)
- 目標時間を超過しても**停止せず延長計測**。
- 延長時はタイム表示が**オレンジ色**に変化。
- 目標到達時のアラートは**一度のみ**（設定ON時）。

### 2.3 自由モード (Free)
- 目標時間に達すると**停止**。
- 停止後、ボタンを押すまで**アラートが鳴り続ける**。

## 3. 自動移行
- ボタンを押すと、現在のセッションを記録し、**反対モードに切り替えて自動スタート**。
- 自由モードのアラート中も同様（STOP → Work自動開始）。

## 4. 記録・履歴
- モード切替時にセッション（種類・経過時間）を自動記録。
- 履歴画面から過去50件の閲覧・削除が可能。

## 5. 設定（永続化）
- Work / Free の目標時間（分・秒）。
- Work目標到達時のアラートON/OFF。
- 全設定は `AppStorage` で永続化。

## 6. 非機能要件
- **プラットフォーム**: iOS (SwiftUI)
- **バージョン管理**: GitHub (`fe_modoki`)
