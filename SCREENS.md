# 画面定義書 - v1.5.0

## 1. メイン画面

```
┌───────────────────────────────┐
│ [履歴]              [設定] │
│                               │
│                               │
│            Work               │  ← モード名（赤 or 緑）
│           05:23               │  ← カウントアップ表示（延長時オレンジ）
│                               │
│                               │
│         ┌───────┐             │
│         │  ⇄    │             │  ← 切替ボタン（通常時）
│         │Free に│             │     反対モードの色で表示
│         │ 切替  │             │
│         └───────┘             │
│                               │
│           v1.5.0              │  ← バージョン表示
└───────────────────────────────┘
```

### 要素一覧

| # | 要素 | 位置 | 説明 |
|---|------|------|------|
| 1 | 履歴アイコン | 左上 | 履歴画面を表示 |
| 2 | 設定アイコン | 右上 | 設定画面を表示 |
| 3 | モード名 | 中央上部 | "Work"（赤）または "Free"（緑） |
| 4 | タイマー表示 | 中央 | MM:SS（延長時はオレンジ色） |
| 5 | メインボタン | 中央下部 | 状態に応じて表示が変化（下記参照） |
| 6 | バージョン | 最下部 | "v1.5.0" |

### メインボタンの状態

| 状態 | アイコン | テキスト | 色 | 動作 |
|------|----------|----------|-----|------|
| 通常（running） | ⇄ | "{反対モード} に切替" | 反対モードの色 | セッション記録→切替→自動開始 |
| アラート中（alerting） | ■ | "STOP" | 赤 | アラート停止→切替→自動開始 |

## 2. 設定画面（Sheet）

| 項目 | 種類 | 範囲 |
|------|------|------|
| Work Minutes | Stepper | 0〜120 |
| Work Seconds | Stepper | 0〜59 |
| Free Minutes | Stepper | 0〜120 |
| Free Seconds | Stepper | 0〜59 |
| Alert at Work target | Toggle | ON/OFF |

## 3. 履歴画面（Sheet）

| 要素 | 説明 |
|------|------|
| セッション一覧 | モード名（色付き）、日付、経過時間 |
| 削除 | スワイプ削除、編集モード対応 |
| 最大件数 | 50件 |
