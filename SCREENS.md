# 画面定義書 - v2.1

## 1. 開始画面

```
┌───────────────────────────────┐
│                               │
│                               │
│    ┌──────┐    ┌──────┐       │
│    │🏋️    │    │🎮    │       │
│    │ Work │    │ Free │       │
│    │スタート│   │スタート│      │
│    └──────┘    └──────┘       │
│    (ワインレッド)  (ミッドナイトブルー)    │
│                               │
│    📅カレンダー  ⚙️設定        │
│                               │
│           v2.1                │
└───────────────────────────────┘
```

- 装飾なし。ボタンは角丸30の正方形（130×130）。
- カレンダーと設定は白テキスト＋アイコン。

```
│          12:34                │  ← 白色、100pt thin mono
│                               │
│                               │
│  ┌─────────────────────────┐  │
│  │       Free へ            │  │  ← 大ボタン（白カプセル）
│  └─────────────────────────┘  │
│                               │
```

```
│                               │
└───────────────────────────────┘
```

## 9. 要素サイズ定数

| 要素 | サイズ・スタイル |
|------|----------------|
| スタートボタン | 130×130, 角丸30 |
| タイマー文字 | 100pt thin monospaced |
| 大ボタン（Next） | maxWidth, padding 20, Capsule |
| 小ボタン（Pause/Finish） | frame(width:100, height:60) |
| モードヘッダー | 24pt bold rounded |
| バージョン | caption2, gray 0.4 opacity |
