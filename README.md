# 作業管理タイマー (Work/Free Timer)

[「１５分チェンジ法」](https://ixy.blog.jp/archives/2366349.html)に基づき、集中力をコントロールするための iOS アプリです。
「好きなこと（Free）」と「やるべきこと（Work）」を交互に繰り返すことで、作業への心理的障壁を下げ、自然な集中状態（フロー）へ導くことを目的としています。

> **Work の延長はフロー状態の証、Free の延長は厳禁。** この非対称なルールがメソッドの核心です。

## 主な機能

- **マルチ画面構成**: 開始・タイマー・完了・カレンダーの4画面で直感的に操作
- **Work（仕事）モード**: 
  - 背景色: **ワインレッド**
  - 目標時間を経過しても停止せず、**明るい緑色**で延長時間をカウントアップ
- **Free（休憩）モード**:
  - 背景色: **ミッドナイトブルー**
  - 目標時間に達すると **0:00 で停止** し、**赤色**のタイマー表示＋アラート鳴動
  - Free の延長は厳禁。1分ごとにリマインド通知で Work 復帰を催促
- **一時停止 / 再開**: 全モードでタイマーの一時停止と再開が可能
- **バックグラウンド対応**:
  - バックグラウンドでもタイマーが正確に進行し、目標到達時に通知
  - Free 超過時は **1分ごと** にリマインド通知
- **ロック画面操作**:
  - 「STOP / 次へ」アクションでアプリを開かずにモード切替
- **月間統計 & 履歴**:
  - カレンダー画面で月間 Work 時間を棒グラフで可視化
  - 各セッションにメモを追加可能

## 使い方

1. **スタート**: Work または Free ボタンでタイマー開始
2. **タイマー中**: 目標時間にフォーカスし、大きな切り替えボタンで次のモードへ
3. **終了**: 「終了」ボタンで Work 合計時間のサマリーを確認
4. **振り返り**: カレンダー画面で過去の成果を確認・メモ

## 開発環境

- **言語**: Swift (SwiftUI)
- **ターゲット**: iOS 17.0+
- **フレームワーク**: Charts, UserNotifications, AudioToolbox
- **ビルド**: `swiftc` または Xcode
