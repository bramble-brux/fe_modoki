# 作業管理タイマー (Work/Free Timer)

「１５分チェンジ法」に基づき、集中力をコントロールするための iOS アプリです。
「好きなこと（Free）」と「やるべきこと（Work）」を交互に繰り返すことで、作業への心理的障壁を下げ、自然な集中状態（フロー）へ導くことを目的としています。

## 主な機能

- **カウントダウンタイマー**: 目標時間からカウントダウンし、残り時間を一目で確認できます。
- **Work（仕事）モード**: 
  - 目標時間を経過しても停止せず、**グリーン（成功色）**で延長時間をカウントアップします。
  - 「どれだけ余分に頑張れたか」を可視化します。
- **Free（休憩）モード**:
  - 目標時間に達すると **0:00 で停止** し、アラートが鳴り続けます。
  - 明確に画面が **赤色** に変化し、休憩の終了を促します。
- **バックグラウンド対応**:
  - アプリを閉じても時間は正確に進み、目標時間に達すると通知が届きます。
- **ロック画面操作**:
  - 通知内のボタンから、アプリを開かずに次のモードへの切り替えとタイマー開始が可能です。
- **統計・履歴**:
  - セッション終了時に今回の Work 合計時間を表示します。
  - 過去 50 件のセッション履歴を保存し、いつでも確認できます。

## 使い方

1. アプリを起動し、最初のモード（Work または Free）を選択します。
2. タイマーが開始されます。
3. 目標時間に達するか、途中で切り替えたい場合は中央の **STOP** ボタンを押します。
4. 自動的に反対のモードに切り替わり、新しいタイマーがスタートします。
5. 作業を完全に終了する場合は、右上の **「終了」** ボタンを押してサマリーを確認します。

## 開発環境

- **言語**: Swift 5.10 (SwiftUI)
- **ターゲット**: iOS 17.0+
- **ビルドツール**: `swiftc` または Xcode
